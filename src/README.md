# Apollo Server Todo

A repository to layout a Todo application with the Apollo framework.

### Install

clone the repository

```
git clone https://github.com/JoseHerminioCollas/apollo-server-todo

```

install the packages
```
yarn
```
### Run

run the server
```
yarn run start
```

test
```
yarn run test:watch
```
